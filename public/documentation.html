Endpoint = Allow new users to join API URL = localhost:8080/users method type = POST query parameters = /users //* must include name, password, email and dob*// request body = a JSON object holding data about user information example: { "name": "Jane Doe", "password": "dog", "email": "jane.doe@gmail.com", "dob": "12/12/1912" } response body = a JSON object holding data about the new user information including an uuid example: { "name": "Jane Doe", "password": "dog", "email": "jane.doe@gmail.com", "dob": "12/12/1912", "id": "6da05320-7589-4dae-8fc4-2468f1667f3b" } 6. Endpoint = allow users to add a favorite movie URL = localhost:8080/users/[id]/addFavorite/[movie title] Method type = PUT query parameters = /users/:id/addFavorite/:movie request body = n/a response body = a JSON object holding data abut the movie added to favorites list example: { "id": "1", "name": "Alicia Millis", "password": "angel", "email": "alicia.millis@gmail.com", "dob": "11/13/1983", "favorites": [ "movie", "Kill Bill" ] } 7. Endpoint = allow user to remove movies from their favourite movie list URL = localhost:8080/users/[id]/removeFavorite/[movie] Method Type = PUT query parameters = /users/:id/removeFavorite/:movie request body = n/a response body = a JSON object holding data about the favorite movie removed example: { "id": "1", "name": "Alicia Millis", "password": "angel", "email": "alicia.millis@gmail.com", "dob": "11/13/1983", "favorites": [ "Kill Bill" ] } 8.
